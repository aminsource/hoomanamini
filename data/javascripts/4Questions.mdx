#### تفاوت undefined , not defined در جاوا اسکریپت چیست؟

در جاوااسکریپت اگر بخواهیم از متغیر که وجود ندارد و تعریف نشده است استفاد کنیم، خطای var name is not defined دریافت می کنیم.
که در این حالت می گوییم متغییر تعریف نشده است یا  not defiend است.

مانند مثال زیر

<div dir="ltr">

```javascript

console.log(y);  // Output: ReferenceError: y is not defined

```
</div>

اما ممکن است متعییر تعریف شده باشد و مقدار دهی نشده باشد که در این حالت undefined است

 مانند مثال زیر:

<div dir="ltr">

```javascript
var x; // declaring x
console.log(x); // output: undefined

```
</div>
